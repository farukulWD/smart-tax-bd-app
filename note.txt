## Onboarding Screen

Create a responsive **3-page onboarding flow** in React Native.

### Layout

* SafeAreaView with a light background.
* Top-right **Skip** button (hidden on the last page).
* Centered illustration section with decorative background circles and a document illustration.
* Title and description centered below the illustration.
* Pagination dots below the description.
* Primary CTA button fixed at the bottom.

### Onboarding Data

```ts
[
  {
    title: "Simplify Your Tax Filing",
    description: "File your income tax return in minutes, right from your phone.",
  },
  {
    title: "Every Income Type Covered",
    description: "From salary to rental to agriculture—we handle every category of income.",
  },
  {
    title: "Track Every Step",
    description: "Follow your filing status and get reminders until it's done.",
  },
]
```

### Behavior

* Support horizontal swipe between pages.
* **Next** advances to the next page.
* On the final page, change the button text to **Get Started**.
* **Skip** navigates directly to the authentication/home screen.
* Hide the **Skip** button on the last page.
* Update pagination to reflect the current page.
* Save onboarding completion using `AsyncStorage` so it only appears on first launch.

### UI

* Minimal, modern, and clean design.
* Large illustration centered at the top.
* Bold heading with supporting description.
* Rounded primary button spanning most of the screen width.
* Active pagination dot is wider and uses the primary color.
* Fully responsive across Android and iOS.
